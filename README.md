# E-Commerce Sales & Customer Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales, customer behavior, product performance, and customer segments using Python, MySQL, and Power BI. The goal is to generate business insights through data cleaning, RFM segmentation, SQL analysis, and interactive dashboards.

## Tools & Technologies

* Python
* Pandas
* NumPy
* MySQL
* Power BI
* GitHub

## Dataset

The project uses e-commerce transaction data containing customer, product, sales, profit, discount, and order information.

## Project Workflow

1. Data Cleaning and Preprocessing using Python
2. Feature Engineering
3. Customer RFM Segmentation
4. Database Design and SQL Analysis
5. Interactive Dashboard Development in Power BI

## Dashboard Pages

### Executive Overview

* Total Sales
* Total Profit
* Total Orders
* Total Customers
* Monthly Sales Trend
* Sales by Region
* Profit by Region

### Product Analysis

* Sales by Category
* Profit by Category
* Top Products by Sales
* Top Products by Profit
* Discount vs Profit Analysis

### Customer Analysis

* Top Customers by Sales
* Customers by Segment
* Revenue by Segment
* Revenue Share by Segment
* Average Revenue per Customer

## Key Insights

* Loyal Customers generate the highest revenue.
* Discounts have a significant impact on profitability.
* A small group of products contributes a large share of total sales.
* Customer segmentation helps identify valuable and at-risk customers.
* Regional performance varies across different markets.
